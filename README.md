# Pizza-Track

## Objetivo
Comprender el concepto de pila y su estructura lógica mediante la implementación de un simulador de gestión de pedidos para una pizzería usando Java.

## Descripción
Pizza-Track es una aplicación en consola que permite registrar pedidos de pizzas y gestionar operaciones de Deshacer (Undo) y Rehacer (Redo) utilizando dos pilas implementadas con listas ligadas.

## Funcionalidades
1. Registrar Pizza
2. Deshacer pedido (Undo)
3. Rehacer pedido (Redo)
4. Mostrar pedido actual
0. Salir

## Estructura del proyecto

- `Pizza.java` → Representa la pizza con nombre y 3 ingredientes.
- `Nodo.java` → Nodo de la lista ligada.
- `PilaPizza.java` → Implementación manual de la pila.
- `GestionPedidos.java` → Maneja las pilas de Undo y Redo.
- `Main.java` → Contiene el menú de interacción con el usuario.

## Ejecución del programa

1. Compilar el proyecto
2. Ejecutar la clase Main
3. Usar el menú en consola para registrar pizzas y probar Undo / Redo.

## Ejemplo de uso

1. Registrar Pizza
2. Deshacer
3. Rehacer
4. Mostrar pedido actual
